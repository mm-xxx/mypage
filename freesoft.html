Visual Studio Code
このHPを作るにあたって使用したテキストエディタ。

VLC media player
ほぼ全ての拡張子の動画が再生できる。変換もできる。

Audacity
録音や音楽ファイルの編集ができる。

GIMP
画像編集ができる。

PictBear
画像編集やお絵描きができる。ちょっと古い？

Photoscape
画像の一括編集や結合、Gifアニメが作れる。

えこでこツール
ほとんどの音楽・動画ファイルからwav、mp3、oggに変換することができる。

AG-デスクトップレコーダー
画面録画できる。

Xmedia Recode
動画のエンコードができる。

pdf_as
PDFの結合等ができるPDF加工ソフト。
